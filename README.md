# BattleSimulator

Este é um programa de RPG de batalha por turnos ambientado na Torre dos Desafios.
O jogo possui uma lista de inimigos predefinidos, e cada vez que você derrota um deles, avança automaticamente para o próximo confronto.

Você começa escolhendo uma classe e entra direto na primeira batalha.
A cada vitória, recebe uma recompensa aleatória para ajudá-lo no combate seguinte.

Continuar lutando até derrotar todos os inimigos e triunfar na torre e triunfar com a vitória, ou Ou fugir a qualquer momento, encerrando o desafio
